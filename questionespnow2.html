ESPNOW 2 문제

코드 설명:

이 코드는 보드B가 보드A에게 1byte의 데이터(123)를 송신하고 보드A는 보드 B로부터 데이터를 수신하는 예제입니다.

  • Board A는 Board B의 MAC 주소를 설정하고, 이 정보를 ESP-NOW 네트워크에 등록한다.
  • Board B로부터 데이터를 수신하면 OnDataRecv 함수가 호출되고 수신된 데이터는 num 변수에 저장되고 시리얼 모니터에 출력된다.
  • Board B는 2초마다 num = 123이라는 데이터를 Board A로 송신한다.
  • Board B는 송신이 완료되었을 때 onDataSent 함수가 호출되고 "성공적으로 송신했음!" 혹은 "송신 실패!!"가 시리얼 모니터에 출력된다.
  • 만약 데이터 송신이 성공적으로 이루어졌다면 status 값이 0이 되고, "성공적으로 송신했음!" 메시지가 시리얼 모니터에 출력된다.
  • 만약 데이터 송신이 실패하면 status 값이 0이 아닌 다른 값이 되고, "송신 실패!!" 메시지가 시리얼 모니터에 출력된다.
