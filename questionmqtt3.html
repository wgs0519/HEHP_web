코드 설명:

  • mytime: LED가 켜진 시간을 기록하기 위한 변수이다.
  • limit: LED가 켜져 있을 시간을 설정하기 위한 변수이다.
  • if (digitalRead(led1) == HIGH) { if (millis() - mytime > limit) { digitalWrite(led1, LOW); LED가 켜져 있는지 확인하고 LED가 켜져 있다면, LED가 켜진 후 경과한 시간이 설정한 limit보다 큰지 확인하고 만약 경과한 시간이 limit를 초과했다면 LED를 끈다.
  • 즉 이 코드는 MQTT 메시지를 통해 LED가 켜진 후 설정된 시간이 지나면 자동으로 LED를 끄는 타이머 역할을 한다.
