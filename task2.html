Task 2

코드 설명:

이 코드는 ESP32의 GPIO 입력 신호를 감지하고, 특정 조건이 충족되면 flag를 얻는 예제입니다.

  • gpio_isr_handler(): GPIO 핀에 변화가 있을 때 인터럽트가 발생하며, trigger 변수를 증가시킵니다.
  • hardware_task2(void pvParameters): FreeRTOS 태스크로, trigger 변수를 모니터링하며, 10,000 이상일 때 메시지를 출력하고 태스크를 종료합니다.
  • setup(): 초기 설정을 수행합니다. 시리얼 통신을 시작하고, GPIO 핀을 입력 모드로 설정하며, 인터럽트를 설정합니다. 또한, FreeRTOS 태스크를 생성합니다.
  • loop(): Arduino의 기본 루프 함수로, 이 코드에서는 비워두고 모든 작업을 FreeRTOS 태스크에서 처리합니다.
